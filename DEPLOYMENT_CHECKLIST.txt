  1. Check YOUTUBE_API_KEY is set correctly in Render
  2. Check YouTube Data API v3 is enabled (console.cloud.google.com)

Solution:
  1. Check GMAIL_PASSWORD is correct (16-char app password, not regular password)
  2. Check 2-Factor authentication is enabled on Gmail
  3. Check GMAIL_USER is: kplayz.official@gmail.com
  4. Try sending test email to see error message

Problem: Website won't load
Solution:
  1. Check Render Dashboard logs
  2. Ensure package.json is at ROOT level (not in subfolder)
  3. Wait 10 minutes (might still be deploying)
  4. Try hard refresh (Ctrl+Shift+Del, clear cache, reload)

Problem: Styling looks broken
Solution:
  1. Clear browser cache (Ctrl+Shift+Del)
  2. Hard refresh (Ctrl+F5 or Cmd+Shift+R)
  3. Try different browser

Problem: Playlists not showing
Solution:
  1. Check playlist IDs are correct (should start with "PL")
  2. Make sure playlists are public (not private)
  3. Check YouTube API has quota left
